# Счетчик на React

Посмотреть в браузере [https://Stanislav-Vasilevich.github.io/counter](https://Stanislav-Vasilevich.github.io/counter).

## Команды для инициализации и запуска:

1. В нужную папку через терминал клонируем проект:

    ```text
    git clone https://github.com/Stanislav-Vasilevich/counter.git
    ```

2. устанавливаем все необходимые зависимости:

    ```text
    yarn
    ```

3. запускаем проект:
    ```text
    yarn start
    ```
